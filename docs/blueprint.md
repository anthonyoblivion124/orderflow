# **App Name**: OrderFlow

## Core Features:

- User Authentication: Secure user login and authentication system with role-based permissions (e.g., admin, manager, viewer).
- PO Management: Create, read, update, and delete purchase orders, with fields for supplier, items, quantities, prices, and dates.
- Supplier Details: Maintain a supplier database with contact information, addresses, and other relevant details.
- Currency Conversion: Ability to manually include the currency rate within the add order page, as each order might be in different currency rate even for same currency. For instance, yesterday orders from Singapore can have currency rate of 120 and today currency rate of orders from Singapore may be different.
- Search and Filter: Filtering, sorting, and searching functionalities to quickly find specific purchase orders or suppliers.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) to convey trust and stability, crucial for a financial application.
- Background color: Light Gray (#F5F5F5), a neutral backdrop that ensures readability and reduces eye strain.
- Accent color: Teal (#009688), offering a contrasting visual cue to highlight important actions and information.
- Clean, sans-serif fonts for clear data presentation and easy readability.
- Simple, consistent icons to aid navigation and represent key functions within the app.
- A clear and structured layout to allow for efficient and logical arrangement of information, facilitating data input and review.
- Subtle transitions and loading animations to provide user feedback and a polished feel.
